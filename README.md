# Threejs

`three.js`是 JavaScript 编写的 `WebGL` 第三方库。
* 使用 `Three.js` 不需要了解太多图形学知识，可以降低学习成本。
* 使用ThreeJS可以用更少的代码实现更多的内容，它屏蔽很多处理细节，提供更多组件，提高开发效率。
* ThreeJS有很高的易用性，但同时也不失灵活性。
